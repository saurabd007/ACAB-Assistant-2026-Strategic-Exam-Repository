# 🎯 ACAB Assistant 2026 Strategic Exam Repository

> **An interactive, single-page repository** designed for candidates preparing for the **Assam Cooperative Apex Bank Ltd. (ACAB) Assistant Recruitment Exam** scheduled for **June 28, 2026**.

---

## 📌 Project Overview

This repository serves as a **data-driven preparation hub**. It synthesizes:
- The latest **2026 fiscal data**
- **RBI monetary policies**
- **Assam-specific cooperative legislation**

into an **interactive dashboard** and **mock exam simulator**.


You Can Access It here: https://saurabd007.github.io/ACAB-Assistant-2026-Strategic-Exam-Repository/

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| **📊 Strategic Dashboard** | Visualizes the 100-mark distribution across the four core sections |
| **📝 125-Question Mock Simulator** | Categorized interface covering General/Financial Awareness, English, Reasoning, and Numerical Ability |
| **🎲 2026 Probability Matrix** | Dynamic charts identifying high-yield topics based on current banking trends and state budget targets |
| **📋 Strategic Briefings** | Condensed insights on "Mission SAKSHAM," RBI's June 2026 "Neutral" stance, and the Assam Cooperative Societies Act updates |

## 🚀 How to Run

This is a **standalone, browser-based application**. No installation required!

1. **Download** the `index.html` file
2. **Open** the file in any modern web browser (Chrome, Firefox, Safari, or Edge)
3. **Start practicing** immediately!

> All dependencies (Tailwind CSS, Chart.js) are loaded via CDN — no local server needed.
