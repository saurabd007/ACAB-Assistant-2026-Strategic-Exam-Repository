ACAB Assistant 2026 Strategic Exam Repository

An interactive, single-page repository designed for candidates preparing for the Assam Cooperative Apex Bank Ltd. (ACAB) Assistant Recruitment Exam scheduled for June 28, 2026.

📌 Project Overview

This repository serves as a data-driven preparation hub. It synthesizes the latest 2026 fiscal data, RBI monetary policies, and Assam-specific cooperative legislation into an interactive dashboard and mock exam simulator.

Key Features

Strategic Dashboard: Visualizes the 100-mark distribution across the four core sections.

125-Question Mock Simulator: A categorized interface covering General/Financial Awareness, English, Reasoning, and Numerical Ability.

2026 Probability Matrix: Dynamic charts identifying high-yield topics based on current banking trends and state budget targets.

Strategic Briefings: Condensed insights on "Mission SAKSHAM," RBI's June 2026 "Neutral" stance, and the Assam Cooperative Societies Act updates.

🚀 How to Run

This is a standalone, browser-based application.

Download the index.html file.

Open the file in any modern web browser (Chrome, Firefox, Safari, or Edge).

No local server or installation is required as all dependencies (Tailwind CSS, Chart.js) are loaded via CDN.

🛠️ Technical Stack

Frontend: HTML5, Tailwind CSS

Visualizations: Chart.js (Canvas-based)

Logic: Vanilla JavaScript for state management and filtering.

⚠️ Disclaimer

This repository is a simulation tool. The data, including "Probable Questions," is generated based on historical patterns and 2026 fiscal projections for academic preparation purposes. It does not guarantee the appearance of specific questions in the actual ACAB examination.
